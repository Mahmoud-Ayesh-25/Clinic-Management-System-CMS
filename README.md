## Clinic Management System

**Clinic Management System** is a full-featured desktop application designed to manage and organize daily operations inside medical clinics in a simple, efficient, and secure way.

The system is developed using **C# and .NET Framework** with **SQL Server** as the database engine. It features a modern **dark gray theme** that provides a clean and comfortable user experience suitable for long working hours.

---

## Features
- Patient management (Add, Edit, Delete, Search)
- Doctor management and assignment to visits and appointments
- Visit management with full patient and doctor linkage
- Payment management with configurable payment methods
- Appointment scheduling system with **real-time notifications**:
  - Notifications when an appointment is approaching
  - Notifications when the appointment time starts
  - Notifications for delayed appointments
- Interactive **Dashboard** with cards displaying summarized data and key statistics
- Settings interface for:
  - Managing payment methods
  - Configuring notifications
  - Editing the currently logged-in user profile
- **Role-based access control system** with fixed and secure permissions
- **Secure password storage** using **Hash + Salt** to protect user credentials
- Clean, user-friendly interface optimized for daily clinic usage
